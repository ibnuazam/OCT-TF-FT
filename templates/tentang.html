Tentang

Deteksi Kelainan pada Makula Retina Mata Menggunakan Transfer Learning dan Fine-Tuning

Sistem ini dibuat untuk prototipe penelitian Skripsi dengan studi kasus klasifikasi dataset Optical Coherence Tomography (OCT).
Program menggunakan server Jupyter Notebook, machine learning framework Tensorflow, API Keras, dan web framework FLask.
Sistem menghasilkan Confusion Matrix dan Grafik Akurasi dan Loss masing-masing pada tahap Transfer Learning dan Fine-Tuning untuk kemudian hasil dari sistem dianalisis lebih lanjut dan menghasilkan ukuran performa Accuracy, Precision, Recall, dan F1-measure.
Hasil penelitian dan dibandingkan dengan penelitian sebelumnya, metode Transfer Learning dikombinasikan dengan Fine-Tuning menghasilkan nilai akurasi yang lebih tinggi yang dicapai lebih cepat dengan epoch yang lebih sedikit.
